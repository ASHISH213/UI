htmldocument    javascripr document

cssDOC allows you to view the Mozilla Developer Network CSS documentation for the selection in your editor. If your selection matches a CSS entity, the documentation page is opened in your default browser. If cssDOC is not able to identify the selection as a CSS entity, the default browser is opened and the Mozilla Developer Network documentation is searched (using a CSS filter).
